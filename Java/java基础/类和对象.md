### **方法**(Methods)
```java
class Class{
    Modifiers Type method(Type parameter, ...){
        // do some stuff with argument or parameter
    }
} 
```
> 方法或构造器重载：方法名相同，参数列表不同

### **参数传递**
#### **传递基本数据类型**
> 传递的是基本数据类型，执行赋值，当方法返回时，对参数值的任何修改都将丢失。

#### **传递引用数据类型**
> 传递的是引用数据类型，赋予引用地址，方法返回时，对参数的更改不会丢失。
