## 接口
```java
interface Interface{
    // private static final
    Object field;

    Object method(Object params...);

    
    default Object method(Object params...){

    }

    static Object method(Object params...){

    }
}
```

```java
class Class implements Interface{
    Object method(Object params...){

    }
}
```
> 接口是对类行为的抽象，抽象类是对类的抽象，类是对对象的抽象
