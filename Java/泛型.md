### **泛型类、接口、方法**
```java
// 泛型类
public class CustomClass<T>{
    private T t;

    CustomClass(T init){
        this.t = init;
    }
    
    public void set(T param){
        this.t = param;
    }
    
    public T get(){
        return t;
    }

    // 对象泛型方法
    public <S> s objm(){
        S s = null;
        return s;
    }

    // 类泛型方法
    public static <U> U clsm(){
        U u = null;
        return u;
    }
}

// 泛型接口
interface CustomInterface<T>{
    T method(T t);
}
```
