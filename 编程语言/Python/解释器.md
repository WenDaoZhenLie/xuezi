## **虚拟环境**
```
PATH = "$vitual_env/Scripts:$PATH"
```
---
### **Venv**
```python
# 创建虚拟环境
$ python3 -m venv .

# 激活
$ ./activate

# 退出
$ ./deactivate

# 安装包
$ pip install package
```

### **Virtualenv**
`pip install virtualenv`
```python
# 创建虚拟环境
$ virtualenv venv

# 激活
$ ./activate

# 退出
$ ./deactivate

# 安装包
$ pip install package
```

### **Poetry**
`pip install poetry`
```python
# 初始化项目
$ poetry init

# 在当前项目目录下创建虚拟环境
$ poetry config virtualenvs.in-project true

# 安装包
$ poetry add package

# 卸载包
$ poetry remove package
```

### **[Conda](https://docs.conda.io/projects/miniconda/en/latest/)**
```python
# 创建虚拟环境
$ conda craete -n venv python=$version

# 激活
$ conda activate venv

# 退出
$ conda deactivate

# 删除虚拟环境
$ conda env remove -n venv
```
