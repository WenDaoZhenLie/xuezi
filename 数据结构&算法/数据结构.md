## **存储结构**

### **顺序存储结构**
```
用一组连续存储单元依次存储数据元素，数据元素之间的逻辑关系是由元素的存储位置来表示
```

### **链式存储结构**
```
用一组任意的存储单元存储数据元素，数据元素之间的逻辑关系是用指针来表示
```
