## ❓ **提问**

### `提问前`：**搜寻答案、阅读手册**
- 🔍 **关键词搜索**: `" "` (精准匹配)
- 🔄 **近义词**: `~`
- 🚫 **排除关键词**: `-`
- ⚡ **模糊匹配**: `*`
- 🌐 **站内搜索**: `site:url`
- 📅 **指定时间范围**: `after:`, `before:`
- 📄 **指定文件类型**: `filetype:`
- 📝 **标题包含指定关键词**: `intitle:`
- 🔍 **正文包含指定关键词**: `intext:`, `allintext:`
- 🔗 **URL包含指定关键词**: `inurl:`, `allinurl:`
