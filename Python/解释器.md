## **虚拟环境**
---
### **venv**
```python
# 创建虚拟环境
$ python3 -m venv .

# 激活
$ ./activate

# 退出
$ ./deactivate

# 安装包
$ pip install package
```

### **virtualenv**
```python
# 创建虚拟环境
$ virtualenv venv

# 激活
$ ./activate

# 退出
$ ./deactivate

# 安装包
$ pip install package
```

### **pipenv**
```python
# 创建虚拟环境
$ pipenv install

# 激活
$pipenv shell

# 退出
$ exit

# 安装包
$ pipenv install package
```
