## **异常**
```python
class Exception(BaseException):
    pass
```

## **抛出异常**
```python
raise Exception("") # from Exception
```

## **异常处理**
```python
try:
    pass
except (Exception,) as e:
    pass/raise
else:
    pass
finally:
    pass
```

## **自定义异常**
```python
class ClassName(Exception):
    pass
```
