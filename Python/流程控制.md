## **条件判断**
> **if...elif...else**
```python
if condition:
    pass
elif condition:
    pass
else
    pass
```

## **循环**
> **for-in**
``` python
for i in iter:
    pass
else
    pass
```

> **while**
``` python
while condition:
    pass
else
    pass
```

## **break**
> 提前结束当前循环

## **continue**
> 跳过本次循环，开始下一次循环
