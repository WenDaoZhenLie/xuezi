## **定义**
```python
def func():
    pass
``` 
---
## **参数**
1. 必选参数
    ```python
    def fun(a,b):
        pass
    ```

2. 默认参数
    ```python
    def fun(a,b=None):
        pass
        # 按顺序传递参数时可以不写参数名
    ```

3. 可变参数
    ```python
    def fun(*valargs):
        for v in valargs:
            pass
        # valargs: list/tuple

    # fun(*list/tuple)
    ```

4. 关键字参数
    ```python
    def fun(**kwargs):
        value = kwargs['key']
        # kwargs: dict

    # fun(**dict)
    ```

5. 命名关键字参数
    ```python
    def fun(a, b, *, c, d):
        pass
        # *后面的参数为命名关键字参数
    ```

## **返回结果**
```python
def fun():
    return x
```
> 不写return时返回None，return可以返回多个值，数据类型为tuple
