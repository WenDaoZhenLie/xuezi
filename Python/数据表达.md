### **整数**
```python
class int(object):
  pass 
```

### **浮点数**
```python
class float(object):
  pass
```

### **字符串**
```python
"""
r" ": 禁止转义
f"{variable}" 格式化字符串
"""
class str(object):
  pass
```

### **布尔值**
```python
class bool(int):
  pass
```

### **空值**
```python
class NoneType(object):
  pass
```

### **变量**
```python
    """
```

### 列表  
```Python
class list(object):
  """
  [,]
  List Comprehensions
  """
  list(iterable=(),/)
``` 

### 元组
```Python
class tuple(object):
  """
  (,)
  ,
  Unpacking
  """
  tuple(iterable=(,/))
```

### **字典**
