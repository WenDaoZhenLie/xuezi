### **整数**
```python
_ = 0
```

### **浮点数**
```python
_ = 0.0
```

### **字符串**
```python
"""
r" ": 禁止转义
f"{variable}" 格式化字符串
"""
_ = ""
```

### **布尔值**
```python
_ = TRUE/FALSE
```

### **空值**
```python
_ = None
```

### **变量**
```python
"""
命名规范：小写，单词之间以下划线分隔
"""
_ = object
```

---

### 列表  
```Python
# 列表生成式
_ = []
_ = list(iterable=(),/)
``` 

### 元组
```Python
# 元组中的元素不可修改
_ = (,)
_ = tuple(iterable=(,/))
```

### **字典**
```Python
_ = {key : value}
_ = dict(key = value)
```

### **集合**
```Python
_ = {,}
_ = set()
```

> 可迭代对象都支持解包
