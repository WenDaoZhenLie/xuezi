## **类(class)**
```python
class CustomClass(object):
    # 构造方法
    '''
    self：创建的实例本身,调用时由解释器传入
    '''
    def __init__(self, *property):
        self.property = property
    
    def __del__(sefl):
        pass
```

## 实例(instance)
```python
obj = CustomClass(*property) # obj.property = property
```
