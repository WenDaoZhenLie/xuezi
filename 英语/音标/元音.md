## **单元音**
```shell
# [ə](中央元音)
央化现象：非重读音节向/ə/靠拢

# [ʌ]
强调的时候发/ʌ/，平时说话发/ə/（诀窍：半张不张，心里想着/ə/）

# [ɑː]
忽略长音记号，竖张口

# [e]
嘴巴微张，舌头放松

# [æ]
舌根稍微上抬，音往鼻腔送，音变现象：1、[æ] + [n]/[m] == [æ + ə + n/m] 2、[æ] + [ŋ] == [eiŋ]

# [i:] & [I]
[i:]: 紧元音，[I]: 松元音（[I]音质 = [i] + [e])

# [u:] & [ʊ]
[u:]: 紧元音(从外往里包)，[ʊ]: 松元音

# [r](特性: 使前面的元音带上[r]色彩(即日化)，[ər]&[ɜːr]被彻底日化，也直接记作[ɚ][ɝ])
舌头回缩，舌尖不碰任何东西
```

## **双元音**
`起始音`->`结尾音`
```shell
# [oʊ]
注意以[ʊ]收尾

# [aʊ]
[a]横向张口（前元音）

# [aɪ] & [eɪ]
收尾音发清楚

# [ɔɪ]
增音现象: [i:][I] -> [j]; [u:],[ʊ] -> [w]

# [ju:]: Uu
半元音字母
```
